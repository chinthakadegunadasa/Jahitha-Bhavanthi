# උපග්‍රන්ථ (Appendices)

## උපග්‍රන්ථය A: පංචස්කන්ධ පද්ධතිමය සිතියම (Systems Mapping)
මෙම වගුව මගින් පංචස්කන්ධයේ එක් එක් ස්තරය නූතන පරිගණක පද්ධති ගෘහ නිර්මාණ ශිල්පය (Computer Architecture) සමඟ සසඳා දක්වයි.

| පංචස්කන්ධ ස්තරය | පද්ධති ඉංජිනේරු සමානතාවය | ක්‍රියාකාරීත්වය |
| :--- | :--- | :--- |
| **රූප ස්කන්ධය** | Hardware Layer / Infrastructure | පද්ධතියේ භෞතික පදනම සහ සංවේදක (Sensors). |
| **වේදනා ස්කන්ධය** | Signal Processing Layer | ලැබෙන දත්තවලට (Input) අදාළ මූලික සංඥා ජනනය. |
| **සංඥා ස්කන්ධය** | Data Labeling & Indexing | ලැබෙන දත්ත හඳුනාගැනීම සහ Metadata ලේබල් කිරීම. |
| **සංස්කාර ස්කන්ධය** | Logic Engine / Processing | දත්ත සැකසීම සහ අනාගත ප්‍රතිදානයන් ගොඩනැගීම. |
| **විඤ්ඤාණ ස්කන්ධය** | Central Awareness (CPU/OS Core) | පද්ධතියේ සමස්ත සවිඥානික බව සහ සම්බන්ධීකරණය. |

## උපග්‍රන්ථය B: හෙළ ස්නායු පුනර්ජනන පර්යේෂණ පදනම
ස්නායු පුනර්ජනනය (Neuro-regeneration) පිළිබඳ පර්යේෂණ සඳහා පාදක වන පුරාණ හෙළ ලේඛන මූලාශ්‍ර:

1.  **වර්ගපූර්ණිකාව:** පඨවි, ආපෝ, තේජෝ, වායෝ ශක්තීන් සෛලීය මට්ටමින් සමතුලිත කිරීමේ ක්‍රමවේද.
2.  **කුමාර තන්ත්‍රය:** ස්නායු පද්ධතියේ වර්ධනය සහ මොළයේ සෛල පද්ධතිය ප්‍රතිසංස්කරණය පිළිබඳ රාවණ යුගයේ වෛද්‍ය විද්‍යාත්මක ප්‍රවේශයන්.
3.  **ස්නායු ප්‍රතිග්‍රාහක (Neural Receptors):** හෙළ වෙදකමේ එන "නිල" විද්‍යාව මගින් ස්නායු පද්ධතියේ දත්ත හුවමාරු වේගය වැඩිදියුණු කරන ආකාරය.

---

## උපග්‍රන්ථය C: තාක්ෂණික පාරිභාෂික ශබ්දකෝෂය (Technical Glossary)

* **De-coupling (විසන්ධි කිරීම):** පද්ධතියේ දත්ත සහ තණ්හාව නමැති එන්ජිම අතර පවතින බැඳීම ඉවත් කිරීම.
* **Infrastructure Sovereignty (යටිතල පහසුකම් ස්වෛරීභාවය):** බාහිර පද්ධතිවල පාලනයෙන් තොරව ස්වාධීනව ක්‍රියාත්මක විය හැකි තාක්ෂණික පදනම.
* **Infinite Loop (අනන්ත ලූපය):** අවිද්‍යාව නිසා පද්ධතිය නැවත නැවතත් උපත කරා (Cycle) ගමන් කිරීමේ දෝෂය.
* **Production Build:** සියලුම දෝෂ නිවැරදි කර අවසාන මුද්‍රණය සඳහා සකස් කළ ග්‍රන්ථයේ පිටපත.
* **Latency (ප්‍රමාදය):** සංඥාවක් ලැබීම සහ එය හඳුනාගැනීම අතර පවතින කාල පරතරය.

## උපග්‍රන්ථය D: පර්යේෂණ මූලාශ්‍ර සහ පරිශීලන ග්‍රන්ථ
* **පාලි ත්‍රිපිටකය:** රතන සූත්‍රය, ඛජ්ජනීය සූත්‍රය සහ පංචස්කන්ධ විභංගය.
* **Madduma Kattadilage Ancestral Records:** හෙළ වෙදකම සහ පුරාණ විද්‍යා පිළිබඳ පවුල් උරුම ලේඛන.
* **Red Hat Enterprise Linux (RHEL) Documentation:** පද්ධති ආරක්ෂණ සහ යටිතල පහසුකම් කළමනාකරණය පිළිබඳ තාක්ෂණික මූලාශ්‍ර.
* **Neuroscience Research Papers:** ස්නායු පුනර්ජනනය සහ මොළයේ ක්‍රියාකාරීත්වය පිළිබඳ නවීන විද්‍යාත්මක ලිපි.
